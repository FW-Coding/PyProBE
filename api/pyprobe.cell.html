class pyprobe.cell.Cell(*, info, procedure=None)#

Bases: BaseModel

A class for a cell in a battery experiment.

Parameters:

  • info (dict) – Dictionary containing information about the cell. The dictionary must contain a ‘Name’ field, other information may include channel number or other rig information.

  • procedure (dict, optional) – Dictionary containing the procedures that have been run on the cell. Defaults to an empty dictionary.
